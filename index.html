Wildcat Bluff Disc Golf Course - Sign Redirects

Redirect the static links used with QR codes on tee signs to allow future edits to target pages

Usage: When requesting a URL that should be redirected specify the target name at the end of the URL using a "sendto" query parameter: ?sendto=
For example: https://wildcatbluffdgc.github.io/redirect.html?sendto=hole2 will redirect to the target URL indicated for "hole2" in redirects.js
