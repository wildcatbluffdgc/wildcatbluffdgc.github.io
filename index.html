Wildcat Bluff Disc Golf Course - Sign Redirects

For use to redirect the static URLs in QR codes on tee signs to allow for changing target pages in the future

To use ../redirect.html :

Specify the target name at the end of the URL using a "sendto" query parameter: ?sendto=targetname


For example: https://wildcatbluffdgc.github.io/redirect.html?sendto=hole2 will redirect to the target URL indicated for hole2 in redirects.js
